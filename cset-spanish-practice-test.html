Part 1: Grammar & Usage

Questions 1–12 test your knowledge of Spanish grammar, including verb forms, pronouns, sentence structure, and language conventions. These questions are representative of the Written Communication domain in Subtest 1.

Question 1Subjunctive Mood
Which sentence correctly uses the present subjunctive?
  • AEspero que el estudiante llega a tiempo.
  • BEspero que el estudiante llegue a tiempo.
  • CEspero que el estudiante llegó a tiempo.
  • DEspero que el estudiante llegará a tiempo.
Correct Answer
B — "llegue"
After expressions of hope, wish, or desire like "esperar que," Spanish requires the subjunctive mood in the subordinate clause. "Llegue" is the present subjunctive form of "llegar." Option A uses indicative (incorrect after "esperar que"), C uses preterite, and D uses future indicative — none of which are appropriate here.
Question 2Preterite vs. Imperfect
Which sentence uses the preterite and imperfect correctly to describe a past interruption?
  • ADormía cuando sonó el teléfono.
  • BDormí cuando sonaba el teléfono.
  • CDormía cuando sonaba el teléfono.
  • DDormí cuando sonó el teléfono.
Correct Answer
A — "Dormía cuando sonó el teléfono."
When describing an action in progress (imperfect) that was interrupted by a completed action (preterite), Spanish uses imperfect + preterite. "Dormía" (I was sleeping — ongoing background action) + "sonó" (rang — a completed interrupting event) is the correct pattern. Option D reverses the logic; options B and C don't correctly capture the interruption structure.
Question 3Ser vs. Estar
Which sentence correctly uses "ser" or "estar"?
  • ALa sopa es muy caliente ahora.
  • BLa sopa está muy caliente ahora.
  • CMi hermana está médica.
  • DEl cielo está azul siempre.
Correct Answer
B — "La sopa está muy caliente ahora."
"Estar" describes temporary states or conditions. A soup being hot right now is a temporary state → "está." Option A is incorrect because "ser caliente" would imply a permanent characteristic. Option C is wrong because professions use "ser" (Mi hermana es médica). Option D is wrong because permanent characteristics use "ser" (El cielo es azul).
Question 4Por vs. Para
Which sentence uses "por" or "para" correctly?
  • ACompré flores para mi madre en el mercado.
  • BEstudié toda la noche para tres horas.
  • CGracias para tu ayuda.
  • DCaminamos para el parque durante una hora.
Correct Answer
A — "Compré flores para mi madre."
"Para" indicates purpose or recipient. Buying flowers for someone (recipient) correctly uses "para." Option B should use "por" (duration of time: estudié por tres horas). Option C should use "por" (thanks for = gracias por). Option D should use "por" (movement through a place or duration: caminamos por el parque).
Question 5Object Pronouns
A teacher wants to say "She explains it to us." Which sentence is correct?
  • AElla nos lo explica.
  • BElla lo nos explica.
  • CElla se nos explica.
  • DElla nos se lo explica.
Correct Answer
A — "Ella nos lo explica."
When two object pronouns occur together in Spanish, the indirect object pronoun (nos = to us) precedes the direct object pronoun (lo = it). The correct order is: indirect + direct, both before the conjugated verb. Option B has the wrong order. Option C incorrectly changes "nos" to "se," which only happens when both pronouns begin with "l" (se lo, se la, etc.). Option D is ungrammatical.
Question 6Conditional Tense
Which sentence correctly uses the conditional tense to express a hypothetical?
  • ASi tengo más tiempo, viajé a España.
  • BSi tuviera más tiempo, viajaría a España.
  • CSi tendría más tiempo, viajara a España.
  • DSi tuve más tiempo, viajaba a España.
Correct Answer
B — "Si tuviera más tiempo, viajaría a España."
In Spanish, contrary-to-fact hypothetical sentences use: "si" + imperfect subjunctive (tuviera) + conditional (viajaría). This is the standard "si clause" structure for unlikely or hypothetical situations. Option A mixes present indicative with preterite. Option C incorrectly puts the conditional in the "si" clause (a common error — the conditional never follows "si"). Option D uses preterite + imperfect, which describes past habitual action, not a hypothetical.
Question 7Reflexive Verbs
Which sentence correctly uses a reflexive construction?
  • ALa madre baña a su hijo. (The mother bathes her son.)
  • BEl niño se baña todos los días. (The child bathes himself every day.)
  • CYo me baño a mi perro. (I bathe myself to my dog.)
  • DEllos se lavan a los platos. (They wash themselves to the dishes.)
Correct Answer
B — "El niño se baña todos los días."
Reflexive constructions require both a reflexive pronoun (se, me, te, nos, os) AND the action performed on the subject itself. "El niño se baña" correctly shows the child performing the action on himself. Option A has no reflexive pronoun (correct, but not reflexive). Options C and D incorrectly combine a reflexive pronoun with a direct object introduced by "a," which is grammatically inconsistent in these contexts.
Question 8Passive Voice
Which sentence correctly uses the passive "se" construction?
  • ASe venden casas en este barrio.
  • BSe vende casas en este barrio.
  • CSe vendieron una casa ayer.
  • DSe venden la casa en este barrio.
Correct Answer
A — "Se venden casas en este barrio."
In the passive "se" construction, the verb must agree in number with the subject (the thing being sold). "Casas" (plural) requires "se venden" (plural verb). Option B uses singular "vende" with plural "casas" — agreement error. Option C uses a singular verb with singular subject but the sentence structure is unusual. Option D uses the definite article "la" before "casa," making it a specific house, which would require a different construction.
Question 9Commands (Imperative)
A teacher wants to tell a student (tú form) "Don't open the book." Which is correct?
  • ANo abres el libro.
  • BNo abra el libro.
  • CNo abras el libro.
  • DNo abre el libro.
Correct Answer
C — "No abras el libro."
Negative tú commands use the present subjunctive form. "Abrir" → present subjunctive tú = "abras," so the negative command is "no abras." Option A uses present indicative (not a command). Option B uses the usted subjunctive form (for negative usted commands). Option D uses the affirmative tú command form, which cannot be used for negative commands.
Question 10Subjunctive — Doubt & Negation
Which sentence correctly uses the subjunctive after a negated verb of thinking?
  • ANo creo que él tiene razón.
  • BNo creo que él tenga razón.
  • CCreo que él tenga razón.
  • DNo creer que él tiene razón.
Correct Answer
B — "No creo que él tenga razón."
When verbs of thinking or believing are negated ("no creer," "no pensar"), they introduce doubt or uncertainty, which triggers the subjunctive in the subordinate clause. "No creo que" → subjunctive (tenga). Option A uses indicative after "no creo que" (a very common learner error). Option C uses "creo que" (affirmative), which normally takes indicative. Option D is not a complete sentence.
Question 11Relative Clauses
Which sentence correctly uses a relative pronoun?
  • AEl libro que leí fue muy interesante.
  • BEl libro quien leí fue muy interesante.
  • CEl libro cual leí fue muy interesante.
  • DEl libro lo que leí fue muy interesante.
Correct Answer
A — "El libro que leí fue muy interesante."
"Que" is the most common relative pronoun and refers to both people and things. For a non-human antecedent (el libro), "que" is the correct choice. "Quien/quienes" refers only to people. "Cual" must be preceded by a definite article (el cual) and is more formal. "Lo que" means "what/that which" and is used when there is no specific antecedent noun — not appropriate here.
Question 12Gender & Number Agreement
Which sentence has correct adjective agreement?
  • ALas profesoras nuevos llegaron temprano.
  • BLas profesoras nuevas llegaron temprano.
  • CLa profesora nuevo llegó temprano.
  • DLos profesores nueva llegaron temprano.
Correct Answer
B — "Las profesoras nuevas llegaron temprano."
Adjectives in Spanish must agree with the noun they modify in both gender and number. "Profesoras" is feminine plural, so the adjective must also be feminine plural: "nuevas." Option A uses masculine plural "nuevos" with feminine "profesoras." Option C uses masculine singular "nuevo" with feminine "profesora." Option D uses feminine singular "nueva" with masculine plural "profesores."

Part 3: Vocabulary & Discourse

Questions 21–25 test vocabulary knowledge, register awareness, and understanding of discourse features in Spanish.

Question 21False Cognates
A student writes: "Estoy muy embarazado por mi error en clase." What is the problem with this sentence?
  • AThere is no problem; the sentence correctly says the student is embarrassed.
  • B"Embarazado" means pregnant, not embarrassed — it is a false cognate.
  • CThe verb "estar" should be replaced with "ser" here.
  • D"Embarazado" is not a real Spanish word.
Correct Answer
B — "Embarazado" means pregnant
"Embarazado/embarazada" is a classic false cognate. It means "pregnant," not "embarrassed." The correct word for embarrassed is "avergonzado/a." This is a critical vocabulary concept for Spanish teachers — students who speak English will frequently make this error. The correct sentence would be: "Estoy muy avergonzado por mi error en clase."
Question 22Register & Formality
A student needs to write a formal letter to a school principal in Spanish. Which greeting is most appropriate?
  • A¡Oye, director!
  • BHola, ¿qué tal?
  • CEstimado señor director:
  • D¿Cómo estás, director?
Correct Answer
C — "Estimado señor director:"
"Estimado/a" (Dear) is the standard formal salutation for professional correspondence in Spanish. It is followed by a colon (:) in Spanish business letters (unlike English, which uses a comma). Options A ("Hey, principal!"), B ("Hi, how's it going?"), and D ("How are you?" using informal "tú" form) are all informal and inappropriate for formal written communication. Knowing register distinctions is an important competency for Spanish teachers.
Question 23Idiomatic Expressions
A student says: "Estoy hecho polvo después del examen." What does this idiomatic expression mean?
  • AI am made of dust.
  • BI am exhausted/worn out.
  • CI am covered in dust from studying.
  • DI destroyed the exam (did very well).
Correct Answer
B — I am exhausted/worn out
"Estar hecho polvo" is a common Spanish idiom meaning to be exhausted, drained, or worn out — like something that has been reduced to dust. It is used after demanding situations (exams, physical work, emotional stress). The expression cannot be interpreted literally. Understanding idiomatic expressions is essential for CSET Spanish candidates, as they appear in authentic text passages.
Question 24Cohesive Devices
Which word or phrase best connects these two sentences coherently? "Los estudiantes trabajaron muy duro. _______, no aprobaron el examen."
  • APor lo tanto
  • BSin embargo
  • CAdemás
  • DEs decir
Correct Answer
B — "Sin embargo" (However)
The two sentences present a contrast: students worked hard (positive), but they didn't pass (negative). "Sin embargo" (however/nevertheless) is the correct adversative connector for this contrast. "Por lo tanto" (therefore) would imply a logical consequence — they worked hard, therefore they failed — which is illogical here. "Además" (furthermore/also) adds information rather than contrasting. "Es decir" (that is/in other words) introduces a clarification or restatement.
Question 25Gerund vs. Infinitive
Which sentence correctly uses the gerund (present participle)?
  • ALeyendo libros es importante para el desarrollo.
  • BMe gusta leyendo libros por la noche.
  • CElla pasó la tarde leyendo novelas.
  • DPara leyendo bien, necesitas practicar.
Correct Answer
C — "Ella pasó la tarde leyendo novelas."
The gerund (leyendo) is correctly used with "pasar + time expression + gerund" to describe how time was spent. This is a standard, idiomatic construction. Option A incorrectly uses the gerund as a subject (Spanish uses the infinitive as a subject: "Leer libros es importante"). Option B incorrectly uses the gerund after "gustar" (should be infinitive: "Me gusta leer"). Option D incorrectly uses the gerund after "para" (should be infinitive: "Para leer bien...").

Question 26Pluperfect Tense
Which sentence correctly uses the pluperfect (pluscuamperfecto) to show that one past action was completed before another?
  • ACuando llegamos, los invitados ya salieron.
  • BCuando llegamos, los invitados ya habían salido.
  • CCuando llegamos, los invitados ya han salido.
  • DCuando llegamos, los invitados ya hubieran salido.
Correct Answer
B — 'los invitados ya habían salido'
The pluperfect (había/habían + past participle) describes an action completed before another past action. 'Habían salido' (had left) correctly precedes the moment of 'llegamos' (we arrived). Option A uses preterite, which doesn't clearly convey prior sequence. Option C uses present perfect (appropriate for recent past, not past-before-past). Option D uses past perfect subjunctive, which requires a conditional or subordinate clause context.
Question 27Regional Variation: Present Perfect vs. Preterite
In standard Castilian Spanish (Spain), which sentence would most naturally describe an action completed earlier today?
  • AEsta mañana fui al mercado.
  • BEsta mañana he ido al mercado.
  • CEsta mañana había ido al mercado.
  • DEsta mañana iré al mercado.
Correct Answer
B — 'Esta mañana he ido al mercado.'
In Spain, the present perfect (he ido) is typically used for actions that occurred within a time frame that includes the present (today, this week). In Latin American Spanish, the preterite (fui) is more commonly used for the same context. This regional distinction is important for Spanish teachers, who may have students from various Spanish-speaking backgrounds. Option C uses pluperfect (before another past event) and Option D uses future — both inappropriate here.
Question 28Subjunctive with Temporal Conjunctions
A teacher wants to say 'Call me when you arrive home.' Which sentence is grammatically correct?
  • ALlámame cuando llegas a casa.
  • BLlámame cuando llegues a casa.
  • CLlámame cuando llegaste a casa.
  • DLlámame cuando llegarás a casa.
Correct Answer
B — 'Llámame cuando llegues a casa.'
When 'cuando' introduces a future or hypothetical event in a command or future context, the subjunctive is required. 'Llegues' (present subjunctive) correctly expresses the future/pending arrival. Option A uses indicative present (appropriate only when describing habitual actions: 'Siempre me llama cuando llega'). Option C uses preterite (past). Option D uses future indicative after 'cuando,' which is a common English-influenced error — Spanish never uses the future indicative after temporal conjunctions like 'cuando.'
Question 29Diminutives
What is the primary function of the diminutive suffix in the utterance: 'Ven aquí, hijito, vamos a leer un cuento'?
  • ATo indicate that the child is physically very small.
  • BTo express affection and endearment toward the child.
  • CTo suggest that the speaker is angry with the child.
  • DTo indicate the child is younger than expected for their grade.
Correct Answer
B — To express affection and endearment
In Spanish, diminutive suffixes (-ito/-ita, -cito/-cita) serve multiple pragmatic functions beyond indicating small size. Most commonly, they convey affection, endearment, or emotional closeness — especially with children, close family members, or pets. In this classroom context, 'hijito' expresses warm, nurturing affection rather than commenting on size. Teachers of Spanish-speaking students should recognize that diminutives carry emotional register information, not just semantic content.
Question 30Verbal Periphrases: acabar de
Which sentence correctly uses 'acabar de + infinitive' to express a recently completed action?
  • AAcabo de terminar mis tareas hace una hora.
  • BAcabé de terminar mis tareas.
  • CAcabo de terminar mis tareas.
  • DAcabo terminar mis tareas.
Correct Answer
C — 'Acabo de terminar mis tareas.'
'Acabar de + infinitive' in the present tense expresses an action completed very recently ('I just finished'). Option A is contradictory — 'acabo de' implies immediacy, which conflicts with 'hace una hora' (an hour ago). Option B uses preterite 'acabé' which simply means 'I finished' without the 'just' nuance. Option D is missing the required 'de' before the infinitive. The periphrasis 'acabar de + infinitive' requires the two-word structure: 'acabo de terminar.'
Question 31Gustar-type Verbs
Which sentence correctly uses the verb 'gustar'?
  • AYo gusto los libros de historia.
  • BMe gustan los libros de historia.
  • CMe gusta los libros de historia.
  • DA mí gusta los libros de historia.
Correct Answer
B — 'Me gustan los libros de historia.'
'Gustar' and similar verbs (encantar, molestar, fascinar, doler) use an inverted structure: the thing that pleases is the grammatical subject, and the person who is pleased is an indirect object. Since 'los libros' is plural, the verb must be plural: 'gustan.' 'Me' is the indirect object pronoun (to me). Option A incorrectly conjugates 'gustar' like a regular transitive verb. Option C uses singular 'gusta' with plural subject 'los libros.' Option D omits the required indirect object pronoun 'me.'
Question 32Definite Article with Feminine Nouns
Which sentence uses the correct definite article?
  • ALa agua del río está fría.
  • BEl agua del río está fría.
  • CLo agua del río está fría.
  • DUn agua del río está fría.
Correct Answer
B — 'El agua del río está fría.'
Feminine singular nouns that begin with a stressed 'a-' or 'ha-' use 'el' (not 'la') in the singular to avoid the awkward vowel sequence 'la agua.' However, the noun remains grammatically feminine: it takes feminine adjectives ('el agua fría'), feminine plural articles ('las aguas'), and feminine pronouns ('ella'). Other examples: el águila, el hacha, el alma, el área. This is a common source of confusion for students who assume 'el' always signals a masculine noun.
Question 33Comparatives of Equality
Which sentence correctly expresses that two things are equal?
  • AJuan es más alto como Pedro.
  • BJuan es tan alto como Pedro.
  • CJuan es igual alto que Pedro.
  • DJuan es tan alto que Pedro.
Correct Answer
B — 'Juan es tan alto como Pedro.'
Equality comparisons in Spanish use the structure: 'tan + adjective/adverb + como' (as...as). 'Tan alto como' = 'as tall as.' Option A incorrectly uses 'como' after 'más' (which requires 'que': 'más alto que'). Option C is ungrammatical ('igual de alto que' would be the correct form). Option D uses 'que' after 'tan,' which creates a result clause ('so tall that'), not a comparison of equality.
Question 34Present Progressive: Appropriate Use
Which sentence correctly uses the present progressive (estar + gerund)?
  • ASiempre estoy bebiendo café por las mañanas.
  • BEn este momento, la maestra está explicando la lección.
  • CEstoy viviendo en California desde hace diez años.
  • D¿Estás entendiendo el problema ahora?
Correct Answer
B — 'En este momento, la maestra está explicando la lección.'
The Spanish present progressive describes an action happening at this exact moment ('en este momento'). Option A describes a habitual action (use simple present: 'siempre bebo'). Option C describes a state/situation that has been ongoing — Spanish uses simple present for this: 'Vivo en California desde hace diez años.' Option D is incorrect because 'entender' is a stative verb (mental state) and typically does not appear in the progressive in standard Spanish ('¿Entiendes el problema?' is preferred).
Question 35Ser vs. Estar + Past Participle
Which pair correctly distinguishes 'ser' (passive action) from 'estar' (resultant state)?
  • ALa puerta fue abierta por el viento. / La puerta está abierta.
  • BLa puerta está abierta por el viento. / La puerta fue abierta.
  • CLa puerta es abierta. / La puerta estaba abierta por Juan.
  • DLa puerta fue abierta. / La puerta fue abierta.
Correct Answer
A — 'fue abierta por el viento' / 'está abierta'
'Ser + past participle' forms the passive voice, describing an action performed by an agent ('fue abierta por el viento' = was opened by the wind). 'Estar + past participle' describes the resulting state after an action ('está abierta' = is open — currently in the open state). This distinction is critical: the 'ser' passive focuses on the action and often includes 'por + agent,' while 'estar' focuses on the condition that results.
Question 36Subjunctive in Adjective Clauses
Which sentence correctly uses the subjunctive because the antecedent is unknown or non-existent?
  • ATengo un estudiante que habla cuatro idiomas.
  • BBusco un intérprete que hable mandarín.
  • CConozco a alguien que sepa arreglar computadoras.
  • DVeo a la doctora que trata a mi madre.
Correct Answer
B — 'Busco un intérprete que hable mandarín.'
When the antecedent (the noun the relative clause refers to) is unknown, non-specific, or non-existent, the subjunctive is used in the adjective clause. In Option B, the speaker is searching for an interpreter they don't yet have — the person may or may not exist, so 'hable' (subjunctive) is correct. Option A uses indicative because the student is known and specific. Option C incorrectly uses subjunctive for a known person ('conozco a alguien' = I know someone specific). Question 37Future of Probability
Which sentence uses the future tense to express conjecture or probability about a present situation?

Correct Answer
B — '¿Cuántos años tendrá esa señora?'
In Spanish, the future indicative can express probability or speculation about present situations — equivalent to 'I wonder...' or 'She must be...' in English. '¿Cuántos años tendrá?' = 'How old could she be? / I wonder how old she is.' Options A, C, and D all use the future tense for actual future events, not conjecture. This 'future of probability' usage is an important feature of Spanish that appears in authentic reading and listening materials.
Question 38Double Negation
Which sentence correctly uses Spanish negative construction?
  • ANo vi a nadie en la sala.
  • BNo vi a alguien en la sala.
  • CVi a nadie en la sala.
  • DVi a no nadie en la sala.
Correct Answer
A — 'No vi a nadie en la sala.'
Spanish requires negative concord (double negation): 'no' before the verb must be paired with a negative word (nadie, nada, nunca, jamás, ninguno) elsewhere in the sentence. 'No vi a nadie' (I didn't see anyone) is grammatically correct and required in standard Spanish. Option B uses 'alguien' (someone) after a negative verb, which creates a different meaning ('I didn't see a specific someone'). Option C omits 'no' before the verb — a sentence cannot begin with 'nadie' in object position without 'no' before the verb in standard Spanish (though preverbal 'nadie' as subject is possible).
Question 39Exceptional Gender

'La mano' (Option B) is correctly feminine despite ending in -o — it is a historical exception. Option A and C both correctly use 'el' with masculine nouns. Question 40Personal 'a'

'Mi hija' is a specific, identifiable person, so 'a mi hija' is correct. Option A: 'leche' is a non-personal thing — no personal 'a.' Option C: 'una película' is a non-personal thing. Question 41Infinitive After Prepositions

  • CSin estudiar, es difícil aprobar.
  • DPara hablando bien, necesitas practicar.
Correct Answer
C — 'Sin estudiar, es difícil aprobar.'

Question 42Demonstrative Adjectives
A teacher is pointing to a book on the student's desk (near the student, not the teacher). Which demonstrative is correct?
  • AAbre este libro.
  • BAbre ese libro.
  • CAbre aquel libro.
  • DAbre uno libro.
Correct Answer
B — 'Abre ese libro.'
Spanish has a three-way demonstrative system based on proximity: 'este/esta/esto' (near the speaker), 'ese/esa/eso' (near the listener), 'aquel/aquella/aquello' (far from both). Since the book is on the student's desk (near the listener, not the speaker/teacher), 'ese' is correct. 'Este' would mean near the teacher. 'Aquel' would mean far from both people. This three-way distinction does not exist in English (only this/that), making it a teaching point for Spanish language learners.
Question 43Superlatives
Which sentence correctly forms an absolute superlative?
  • AEsta comida es muy deliciosa.
  • BEsta comida es deliciosa.
  • CEsta comida es deliciosísima.
  • DEsta comida es la más deliciosa.
Correct Answer
C — 'Esta comida es deliciosísima.'
Spanish has two ways to express extreme degree: the relative superlative ('la más deliciosa' = the most delicious, comparing to others) and the absolute superlative ('-ísimo/a' = extremely delicious, no comparison). 'Deliciosísima' is the absolute superlative, formed by adding -ísimo/a to the adjective stem. This is distinct from 'muy + adjective' (Option A), which is simply 'very.' Option D is the relative superlative. Knowing both forms is important for reading authentic Spanish texts and for teaching Spanish.
Question 44Formal Commands (Usted)

  • BHable más despacio, por favor.
  • CHablas más despacio, por favor.
  • DHablar más despacio, por favor.
Correct Answer
B — 'Hable más despacio, por favor.'
Formal (usted) affirmative commands use the present subjunctive form of the verb. For 'hablar,' the usted subjunctive is 'hable.' Option A uses the informal tú affirmative command form. Option C uses the present indicative tú form (not a command). Option D uses the infinitive (not a command). Knowing the distinction between formal and informal commands is essential for Spanish teachers, who must model appropriate register in the classroom.
Question 45Spelling Changes in Preterite
Which preterite form contains a spelling error?
  • AYo busqué el archivo perdido.
  • BYo llegué tarde a la reunión.
  • CYo empezé a estudiar.
  • DYo toqué la guitarra anoche.
Correct Answer
C — 'empezé' is incorrect; should be 'empecé'
Verbs ending in -zar change 'z' to 'c' before 'e' in the preterite yo form: empezar → empecé (not empezé). This spelling change preserves the original pronunciation — 'ze' does not exist in standard Spanish spelling. Similarly: comenzar→comencé, cruzar→crucé. Verbs ending in -car change 'c' to 'qu' before 'e': buscar→busqué (Option A, correct). Verbs ending in -gar add 'u' after 'g' before 'e': llegar→llegué (Option B, correct). Tocar→toqué (Option D, correct).

Knowing antonym pairs is important for vocabulary instruction and for understanding contrast in reading passages ('the program strengthened reading skills while weakening reliance on memorization').
Question 72Register: Tú vs. Usted
A new student's parent enters the classroom. The teacher does not know them. Which greeting shows appropriate professional register?
  • AHola, ¿cómo estás? Soy la maestra Rodríguez.
  • B¡Oye! ¿Qué tal? Soy la maestra Rodríguez.
  • CBuenos días. ¿En qué le puedo ayudar?
  • D¿Qué onda? Soy la maestra Rodríguez.
Correct Answer
C — 'Buenos días. ¿En qué le puedo ayudar?'
In a professional context with an unknown adult, 'usted' (formal address) is required. Option C uses 'le' (indirect object, usted form) and a standard formal greeting — appropriate for an unknown parent. Options A and B use 'tú' forms ('estás,' '¿qué tal?' in informal context), which are too casual for an unknown adult in a professional setting. Option D ('¿Qué onda?') is very informal slang. Spanish teachers must model appropriate register and teach students about 'tú/usted' distinctions for professional and social contexts.
Question 73Word Families
Which word belongs to the same word family as 'enseñar' (to teach)?
  • AEnsayo
  • BEnseñanza
  • CSeñalar
  • DEnsanchar
Correct Answer
B — Enseñanza (teaching/instruction)
'Enseñanza' (teaching, education, instruction) is directly derived from 'enseñar' (to teach) — they share the same root. Word family relationships: enseñar (verb) → enseñanza (noun) → maestro/profesor (agent noun, different root). 'Ensayo' (essay/rehearsal) comes from a different root ('ensayar'). 'Señalar' (to point out/signal) shares the 'señal' root but is not from 'enseñar.' 'Ensanchar' (to widen) has a different etymology entirely. Teaching word families helps students expand vocabulary systematically.
Question 74Prefix Meaning
A student encounters the word 'bilingüe' for the first time. Knowing that the prefix 'bi-' means 'two,' the student can infer that 'bilingüe' means:
  • Aa person who speaks beautifully
  • Ba person or thing involving two languages
  • Ca person who is very intelligent
  • Da formal document in two copies
Correct Answer
B — involving two languages
The prefix 'bi-' means two (from Latin). 'Lengua' means language or tongue. Therefore, 'bilingüe' = two-languaged, involving two languages. This morphological reasoning strategy — using knowledge of prefixes to decode unfamiliar words — is an important vocabulary development strategy explicitly included in California ELA/ELD standards. Other examples: trilingüe (three languages), monolingüe (one language), multilingüe (many languages).
Question 75Suffix Meaning
A student sees the word 'hispanohablante.' Knowing that '-hablante' comes from 'hablar' (to speak), what does 'hispanohablante' most likely mean?
  • ASomeone who studies Hispanic culture
  • BA Spanish speaker / someone who speaks Spanish
  • CA person who lives in a Hispanic neighborhood
  • DA textbook about the Spanish language
Correct Answer
B — A Spanish speaker
'Hispano-' refers to Hispanic/Spanish, and '-hablante' (from 'hablar,' to speak) means speaker. Therefore, 'hispanohablante' = Spanish speaker. This morphological pattern is productive in Spanish: anglohablante (English speaker), francohablante (French speaker), arabohablante (Arabic speaker). Explicit instruction in morphology — prefixes, suffixes, and roots — is one of the most effective vocabulary development strategies, especially for academic and technical vocabulary.
Question 76Transition Words: Causation
Which transition word BEST introduces a cause or reason?
  • ASin embargo
  • BAdemás
  • CPuesto que
  • DNo obstante
Correct Answer
C — Puesto que (since/because)
'Puesto que' introduces a cause or reason (since/given that/because). Other causal connectors: 'porque,' 'ya que,' 'dado que,' 'debido a que.' 'Sin embargo' and 'no obstante' signal contrast (however/nevertheless). 'Además' signals addition (furthermore/also). Knowing the logical relationships encoded by different transition words is essential for both reading comprehension (understanding how ideas connect) and writing instruction (teaching students to organize their arguments).
Question 77Transition Words: Concession
Which sentence correctly uses a concessive connector?
  • AEstudié mucho; por lo tanto, suspendí el examen.
  • BAunque estudié mucho, suspendí el examen.
  • CEstudié mucho, es decir, suspendí el examen.
  • DEstudié mucho; además, suspendí el examen.
Correct Answer
B — 'Aunque estudié mucho, suspendí el examen.'
'Aunque' (although/even though/even if) is a concessive connector — it acknowledges a circumstance while asserting that it didn't produce the expected result. 'Aunque estudié mucho, suspendí' = Although I studied a lot, I failed — the expected result (passing) was not achieved. Option A ('por lo tanto') incorrectly implies studying caused failure. Option C ('es decir') incorrectly implies the second clause restates or explains the first. Option D ('además') incorrectly adds information rather than creating concession.
Question 78Figurative Language
Which sentence contains a simile (comparación)?
  • ALa maestra es un volcán cuando se enoja.
  • BLa maestra explica las matemáticas claramente.
  • CLa voz de la maestra es suave como la seda.
  • DLa maestra tiene muchos estudiantes este año.
Correct Answer
C — 'La voz de la maestra es suave como la seda.'
A simile makes an explicit comparison using 'como' (like/as) or 'tal como.' 'Suave como la seda' (smooth/soft as silk) is a simile. Option A is a metaphor — it says the teacher IS a volcano (direct equation, no 'como'). Options B and D are literal statements, not figurative language. The distinction between metaphor (direct equation) and simile (comparison using 'como') is an important literary device for both reading and writing instruction.
Question 79Academic Vocabulary
A teacher asks students to 'analizar los argumentos del autor.' What is the student expected to do?
  • ACopy the author's arguments word for word
  • BBreak down and examine the author's arguments in detail
  • CAgree with all of the author's arguments
  • DTranslate the author's arguments into English
Correct Answer
B — Break down and examine the arguments in detail
'Analizar' (to analyze) is a high-frequency academic verb meaning to examine something closely by breaking it into components and studying how they work together. In academic contexts, analysis involves identifying claims, evidence, reasoning, and assumptions. Other important academic verbs Spanish teachers should know: comparar (compare), contrastar (contrast), sintetizar (synthesize), evaluar (evaluate), argumentar (argue), inferir (infer). These are the verbs of academic thinking across all subject areas.
Question 80Connotation
The words 'delgado,' 'flaco,' and 'esmirriado' all refer to a thin person. Which has the most negative connotation?
  • ADelgado
  • BFlaco
  • CEsmirriado
  • DAll three are equally neutral.
Correct Answer
C — Esmirriado
These three words exist on a connotation spectrum. 'Delgado' is neutral or slightly positive (slim, slender — often considered attractive). 'Flaco' is neutral to mildly negative (thin, skinny — more informal, can imply being too thin). 'Esmirriado' is clearly negative and informal — it means scrawny, puny, or pathetically thin, often used to mock or criticize. Teaching connotation — the emotional and evaluative associations of words beyond their denotative meaning — is essential for developing reading comprehension and writing precision in Spanish.

Question 81Spanish Vowel System
How does the Spanish vowel system differ from English in a way relevant to oral communication instruction?
  • ASpanish has more vowel sounds than English.
  • BSpanish has five pure, consistent vowel sounds; English has more vowels and many diphthongs.
  • CSpanish vowels change in quality depending on stress, just like English.
  • DSpanish and English have identical vowel systems.
Correct Answer
B — Spanish has five pure, consistent vowel sounds.
Spanish has five vowel phonemes (a, e, i, o, u), each with a single, consistent pronunciation regardless of position or stress — they are 'pure' vowels. English has approximately 14-15 vowel sounds, many of which are diphthongs (vowels that shift in quality during pronunciation). This is why Spanish speakers learning English often struggle with vowel sounds, and why English-speaking students of Spanish must be trained to avoid diphthongizing Spanish vowels (e.g., saying 'no' with an /ow/ sound instead of a pure /o/).
Question 82Spanish Consonants: r vs. rr
What is the phonological distinction between the single 'r' and double 'rr' in Spanish?
  • AThey are pronounced identically; 'rr' is only a spelling convention.
  • BSingle 'r' (except at the start of a word) is a tap; 'rr' (and initial 'r') is a trill.
  • C'rr' is always silent in standard Spanish pronunciation.
  • DSingle 'r' is always trilled; 'rr' is a tap sound.
Correct Answer
B — Single 'r' is a tap; 'rr' and initial 'r' are trills.
In standard Spanish, the single 'r' in the middle of a word is a tap (alveolar flap, like the 'd' in American English 'ladder'). The double 'rr' and 'r' at the beginning of a word are trills (multiple vibrations of the tongue tip against the alveolar ridge). This distinction is phonemic — it changes meaning: 'pero' (but) vs. 'perro' (dog); 'caro' (expensive) vs. 'carro' (car). Spanish teachers must understand this distinction to give students accurate feedback on pronunciation.
Question 83Regional Variation: Seseo vs. Distinción
In most of Latin America and the Canary Islands, speakers pronounce 'c' (before e/i) and 'z' the same as 's.' This feature is called:
  • AYeísmo
  • BVoseo
  • CSeseo
  • DAspiración
Correct Answer
C — Seseo
'Seseo' is the pronunciation feature in which /s/, /c/ (before e/i), and /z/ are all pronounced as /s/. It is the standard in Latin America, the Canary Islands, and parts of Andalusia. In contrast, standard Castilian Spanish (central and northern Spain) uses 'distinción,' where 'c' (before e/i) and 'z' are pronounced as /θ/ (the 'th' sound in English 'think'). Neither is 'wrong' — they are regional varieties of equal linguistic prestige. 'Yeísmo' refers to the merging of 'll' and 'y' sounds. 'Voseo' is the use of 'vos' instead of 'tú.'
Question 84Regional Variation: Yeísmo
What is 'yeísmo' and where is it commonly found?
  • AThe use of 'vos' instead of 'tú' in Argentina and Uruguay
  • BThe merging of 'll' and 'y' into a single sound in most of the Spanish-speaking world
  • CThe aspiration of 's' at the end of syllables in Caribbean Spanish
  • DThe pronunciation of 'v' and 'b' as the same sound
Correct Answer
B — The merging of 'll' and 'y' into a single sound.
'Yeísmo' is the feature by which 'll' and 'y' are pronounced identically — typically as /ʝ/ or /ʒ/. It is now dominant in most of the Spanish-speaking world, including all of Latin America and most of Spain. As a result, 'pollo' (chicken) and 'poyo' (stone bench) are homophones for most speakers. Traditional 'distinción' (maintaining a difference between 'll' and 'y') survives in some areas of Spain and the Andes. Teachers should recognize this so they can explain why students may confuse the spellings 'll' and 'y.'
Question 85Stress Patterns
Which statement about Spanish stress patterns is correct?
  • AAll Spanish words are stressed on the last syllable.
  • BStress in Spanish is unpredictable and must be memorized for every word.
  • CWords ending in a vowel, -n, or -s are typically stressed on the second-to-last syllable; others on the last syllable; written accents mark exceptions.
  • DThe written accent mark always indicates that a word is stressed on the first syllable.
Correct Answer
C — Standard stress rules with written accents for exceptions.
Spanish has predictable stress rules: words ending in a vowel, 'n,' or 's' are normally stressed on the penultimate (second-to-last) syllable (mesa, hablan, casas). Words ending in any other consonant are stressed on the final syllable (hablar, ciudad). Written accent marks indicate stress that deviates from these rules (café, árbol, canción) or distinguish homophones (si/sí, se/sé). Teaching these rules explicitly helps students read aloud accurately and spell correctly.
Question 86Intonation
How does intonation distinguish a yes/no question from a statement in spoken Spanish?
  • ASpanish uses a different verb tense for questions.
  • BSpanish yes/no questions use rising intonation at the end; statements typically fall.
  • CSpanish questions always begin with an interrogative word like '¿qué?' or '¿cómo?'
  • DSpanish uses a special question particle, similar to Japanese 'ka.'
Correct Answer
B — Questions use rising intonation; statements fall.
In spoken Spanish, yes/no questions (¿Vienes?) are typically distinguished from statements (Vienes.) primarily through intonation — a rising pitch contour at the end of the question vs. a falling contour for statements. Unlike English, Spanish does not require subject-verb inversion for questions (though it can occur). This makes prosody (rhythm, stress, and intonation) an essential component of oral communication that teachers must explicitly model and teach, especially to students whose home language has different intonation patterns.
Question 87Code-Switching: Definition
What is 'code-switching' in the context of bilingual communication?
  • ATranslating a text from Spanish to English word by word
  • BThe practice of alternating between two or more languages or language varieties within a single conversation or utterance
  • CUsing formal Spanish in academic settings and informal Spanish at home
  • DTeaching students to correct their grammar errors through self-monitoring
Correct Answer
B — Alternating between two or more languages within a conversation
Code-switching is the linguistically sophisticated practice of moving between languages or dialects within a conversation, a sentence, or even a phrase. It is a natural, rule-governed behavior in bilingual communities — not a sign of confusion or language deficiency. Examples: 'Fui al store porque necesitaba leche.' Research by linguists including Ana Celia Zentella has shown that code-switching follows systematic patterns. Spanish teachers should understand code-switching to respond appropriately and knowledgeably in multilingual classroom contexts.
Question 88Code-Switching: Pedagogical Implications
A teacher hears a bilingual student switch from Spanish to English mid-sentence during a class discussion. What is the most linguistically informed response?
  • AImmediately correct the student and require them to speak only Spanish.
  • BRecognize code-switching as a natural bilingual behavior; use the moment to discuss language choice and context.
  • CRemove the student from the bilingual program for insufficient Spanish proficiency.
  • DIgnore all instances of code-switching to avoid embarrassing the student.
Correct Answer
B — Recognize it as natural bilingual behavior and use it as a teaching moment.
Code-switching is a sophisticated linguistic behavior that indicates bilingual proficiency, not a deficiency. Research consistently shows that bilingual speakers code-switch strategically for various communicative purposes (filling lexical gaps, expressing solidarity, emphasizing a point). Immediately correcting or punishing code-switching is linguistically uninformed and can harm students' confidence and language development. The pedagogically appropriate response is to acknowledge it as natural while also helping students develop awareness of appropriate register and context for each language.
Question 89Features of Spoken vs. Written Spanish
Which feature is MORE characteristic of spoken Spanish than written Spanish?
  • AComplete, well-formed sentences with explicit subjects
  • BExtensive use of subordinate clauses with subjunctive
  • CFrequent ellipsis (omission of elements understood from context) and fillers like 'este... o sea...'
  • DUse of passive voice constructions
Correct Answer
C — Ellipsis and fillers like 'este... o sea...'
Spoken language is characterized by features absent in formal writing: ellipsis (dropping subjects and objects clear from context), discourse fillers ('este,' 'o sea,' 'bueno,' 'pues'), false starts, repetitions, and informal vocabulary. Written Spanish, especially academic writing, uses more complex syntax, explicit subjects, formal connectors, and avoids fillers. Understanding these distinctions helps teachers design oral and written language activities appropriately and helps students understand why the same information sounds different in spoken vs. written form.
Question 90Listening Comprehension Strategies
Which strategy is most effective for improving listening comprehension in a second language?
  • ATrying to understand and translate every single word before moving on
  • BUsing context clues, prior knowledge, and focusing on key words to construct meaning
  • COnly listening to content that contains no unfamiliar vocabulary
  • DReading the transcript silently instead of listening
Correct Answer
B — Using context, prior knowledge, and key words
Effective listeners use top-down processing (using prior knowledge, context, and prediction) alongside bottom-up processing (decoding individual words and sounds). Trying to translate every word (Option A) breaks the flow of comprehension and is ineffective. Exposing learners only to familiar vocabulary (Option C) limits development — Stephen Krashen's comprehensible input hypothesis suggests language acquisition requires input that is slightly beyond current level (i+1). Option D replaces listening with reading, bypassing the skill being developed. Good listeners tolerate ambiguity and focus on overall meaning.
Question 91Comprehensible Input (Krashen)
Stephen Krashen's 'Input Hypothesis' (i+1) suggests that language acquisition occurs best when:
  • AInput is perfectly matched to the learner's current level with no new elements.
  • BInput is slightly above the learner's current level, with enough context to make it understandable.
  • CLearners study grammar rules explicitly before hearing any oral input.
  • DInput is entirely in the target language with no support in the first language.
Correct Answer
B — Input slightly above current level, made understandable by context.
Krashen's Input Hypothesis states that language acquisition advances when learners are exposed to 'comprehensible input' at a level just beyond their current competence (i+1, where 'i' = current level). The '+1' challenge, made comprehensible through context, visuals, gestures, and prior knowledge, promotes acquisition. Option A describes input that is not challenging enough for growth. Option C describes explicit grammar instruction, which Krashen distinguishes from acquisition. Option D may create anxiety and reduce comprehensibility if there is no contextual support. This theory underlies many immersion and communicative language teaching approaches.
Question 92Affective Filter
According to Krashen's Affective Filter Hypothesis, language acquisition is most effective when:
  • AStudents are frequently corrected to ensure accuracy.
  • BStudents feel anxious because stress improves focus and retention.
  • CThe learning environment is low-anxiety, motivating, and builds self-confidence.
  • DGrammar rules are explicitly taught before any oral practice.
Correct Answer
C — Low-anxiety, motivating, confidence-building environment.
Krashen proposes that a psychological 'affective filter' — composed of anxiety, motivation, and self-confidence — can block or allow language input to reach the language acquisition device. When anxiety is high, the filter is 'up' and input doesn't lead to acquisition even if it's comprehensible. When the environment is supportive and motivating, the filter is 'low' and acquisition occurs more readily. This has direct implications for classroom practice: creating safe, encouraging learning environments is not just good pedagogy but a prerequisite for language acquisition.
Question 93Oral Language Development
In first-language oral language development, children typically acquire which structure last?
  • ABasic subject-verb-object sentences
  • BSimple negation (no + verb)
  • CComplex subordinate clauses and passive voice
  • DSingle-word utterances (holophrases)
Correct Answer
C — Complex subordinate clauses and passive voice
First language acquisition follows a predictable developmental sequence. Children begin with single words (holophrases), then two-word combinations, then simple sentences with basic word order (S-V-O), and gradually acquire more complex structures. Complex syntax (embedded clauses, passive voice, subjunctive) develops later — some structures continue developing into adolescence. Understanding this sequence helps Spanish teachers recognize developmentally appropriate language use and set realistic expectations for students' oral production at different ages and proficiency levels.
Question 94Academic Oral Language
Which feature most distinguishes academic oral language from conversational oral language?
  • AAcademic oral language is always louder and more formal in volume.
  • BAcademic oral language uses more precise, content-specific vocabulary and more complex syntax.
  • CConversational language uses more vocabulary words than academic language.
  • DAcademic language is always written, never spoken.
Correct Answer
B — More precise vocabulary and complex syntax.
Academic oral language (used in class discussions, presentations, formal debates) differs from conversational language in several key ways: it uses domain-specific and technical vocabulary, more complex syntactic structures (subordinate clauses, passive voice, nominalization), explicit cohesion markers, and a more formal register. Conversational language relies more on context, shared knowledge, and informal vocabulary. California ELD standards explicitly distinguish between conversational and academic language proficiency, and teachers must develop both in their students.
Question 95Pragmatics: Speech Acts
A student says 'Hace mucho frío aquí' (It's very cold in here) to the teacher. In pragmatic terms, this might be functioning as:
  • AA simple meteorological observation
  • BAn indirect request to close the window or turn up the heat
  • CA complaint about the school's budget
  • DA question about the thermometer reading
Correct Answer
B — An indirect request
Pragmatics is the study of how context shapes the interpretation of language. 'Hace mucho frío aquí,' while literally a statement about temperature, can function as an indirect speech act — a polite, indirect way of requesting an action (close the window, turn on the heat). This is especially common in Spanish-speaking cultures where direct requests to authority figures can be seen as rude. Teachers need pragmatic awareness to interpret student communication correctly and to teach students how to make appropriate requests in academic settings.
Question 96Non-Verbal Communication
A teacher working with Spanish-speaking families should know that compared to some Northern European cultures, many Latin American cultures tend to use:
  • ALess eye contact and greater physical distance in conversation
  • BMore physical proximity, touch, and eye contact in face-to-face conversation
  • CExclusively formal register in all social interactions
  • DSilence as the primary affirmation signal
Correct Answer
B — More physical proximity, touch, and eye contact.
Proxemics (use of physical space) and haptics (use of touch) vary significantly across cultures. Many Latin American cultures use closer physical proximity, more touch (including greetings with kisses or embraces), and sustained eye contact as signs of warmth, respect, and engagement. Northern European and Anglo-American cultures tend toward greater physical distance and less touch in professional contexts. Teachers who are unaware of these differences may misinterpret student or family behavior. Culturally responsive teaching includes awareness of non-verbal communication norms.
Question 97Dialect vs. Accent
What is the key distinction between a 'dialect' and an 'accent'?
  • AA dialect refers only to writing; an accent refers only to speech.
  • BAn accent refers to pronunciation differences; a dialect includes vocabulary, grammar, and pronunciation differences.
  • CDialects are always considered inferior to standard language varieties.
  • DAn accent is a language disorder requiring speech therapy.
Correct Answer
B — Accent = pronunciation; Dialect = pronunciation + vocabulary + grammar.
An 'accent' refers specifically to pronunciation differences from another variety (e.g., a Californian Spanish speaker has a different accent than a Cuban Spanish speaker). A 'dialect' is broader — it includes differences in vocabulary (lexical variation), grammar (syntactic variation), and pronunciation. All speakers of all languages have both an accent and a dialect. No dialect is linguistically inferior to any other; prestige dialects gain status through social and political factors, not linguistic complexity. Spanish teachers must hold this view to teach in a culturally affirming way.
Question 98Clarification Strategies
Which response best models an effective clarification strategy in oral communication?
  • ARepeating the misunderstood statement louder and more slowly.
  • BSaying '¿Puedes repetir eso, por favor? No entendí la última parte.'
  • CSwitching to English to avoid misunderstanding.
  • DNodding and pretending to understand to avoid awkwardness.
Correct Answer
B — '¿Puedes repetir eso, por favor? No entendí la última parte.'
Effective clarification requests are specific (identifying what was not understood — 'la última parte'), polite, and direct. This models good communication behavior for students. Option A (repeating louder) does not address the comprehension problem and can be perceived as condescending. Option C (switching languages) may be appropriate sometimes but is not a clarification strategy. Option D (pretending to understand) impedes communication and models dishonest behavior. Teaching students to ask for clarification explicitly is an important communicative competence in Spanish oral language development.
Question 99Features of Oral Academic Discourse
Which behavior demonstrates oral academic language proficiency in Spanish?
  • AUsing only short, simple sentences to ensure clarity
  • BAvoiding technical vocabulary to make speech more accessible
  • CUsing discourse markers like 'en primer lugar,' 'por otro lado,' and 'en conclusión' to organize ideas
  • DSpeaking as fast as possible to demonstrate fluency
Correct Answer
C — Using discourse markers to organize ideas
Oral academic proficiency includes the ability to organize extended discourse using appropriate markers: 'en primer lugar' (first of all), 'por otro lado' (on the other hand), 'sin embargo' (however), 'en conclusión' (in conclusion). These markers signal the logical structure of an argument and are characteristic of academic speech and formal presentations. Simple sentences (A) and avoiding technical vocabulary (B) may indicate lower proficiency, not higher. Speaking rate (D) is not a reliable indicator of proficiency — clarity, accuracy, and complexity are more meaningful measures.
Question 100Oral Fluency Assessment
When assessing oral fluency in Spanish, a teacher should prioritize:
  • AGrammatical accuracy as the only measure of language proficiency
  • BA balance of fluency, accuracy, complexity, and communicative effectiveness
  • CThe number of English words avoided during the task
  • DWhether the student's accent matches standard Castilian Spanish
Correct Answer
B — A balance of fluency, accuracy, complexity, and communicative effectiveness
Oral language assessment should consider multiple dimensions: fluency (flow without excessive pausing), accuracy (grammatical correctness), complexity (variety of structures and vocabulary), and communicative effectiveness (does the message get across?). Focusing on accuracy alone (A) penalizes students who are communicatively competent but still developing grammar. Counting avoided English words (C) is not a valid measure. Requiring a Castilian accent (D) is linguistically uninformed and culturally biased — all regional accents are linguistically valid. Holistic, multi-dimensional assessment reflects current best practices in language education.
