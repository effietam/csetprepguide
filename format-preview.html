Question 1 Subjunctive Mood
Which sentence correctly uses the present subjunctive?
A
Espero que el estudiante llega a tiempo.
B
Espero que el estudiante llegue a tiempo.
C
Espero que el estudiante llegó a tiempo.
D
Espero que el estudiante llegará a tiempo.
Correct Answer
B — "llegue"
After expressions of hope or desire like "esperar que," Spanish requires the subjunctive mood. "Llegue" is the present subjunctive form of "llegar." Options A (indicative), C (preterite), and D (future) are all incorrect here.
Question 2 Ser vs. Estar
Which sentence correctly uses "estar"?
A
La sopa es muy caliente ahora.
B
La sopa está muy caliente ahora.
C
Mi hermana está médica.
D
El cielo está azul siempre.
Correct Answer
B — "La sopa está muy caliente ahora."
"Estar" describes temporary states. A soup being hot right now is temporary → "está." Professions use "ser" (mi hermana es médica). Permanent characteristics use "ser" (el cielo es azul).
Question 3 False Cognates
A student writes: "Estoy muy embarazado por mi error." What is wrong?
A
Nothing — the sentence correctly says the student is embarrassed.
B
"Embarazado" means pregnant, not embarrassed — it is a false cognate.
C
"Estar" should be replaced with "ser" here.
D
"Embarazado" is not a real Spanish word.
Correct Answer
B — "Embarazado" means pregnant
"Embarazado/a" is a classic false cognate meaning pregnant, not embarrassed. The correct word for embarrassed is "avergonzado/a." This is one of the most important false cognates for Spanish teachers to know and teach explicitly.
